# ⌨️ Typing Speed Test

An interactive typing speed test game built with Vanilla JavaScript to improve typing speed and accuracy.

---

## ✨ Features

- Multiple difficulty levels
- Dynamic word generation
- Countdown timer system
- Score tracking using LocalStorage
- Responsive and simple UI

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
